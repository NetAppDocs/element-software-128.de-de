= Rüsten Sie Ihre vSphere-Komponenten für ein NetApp SolidFire -Speichersystem mit dem Element-Plug-in für vCenter Server auf.
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Beim Upgrade von VMware vSphere-Komponenten sind zusätzliche Schritte erforderlich. Diese Schritte sind für Systeme mit dem Element-Plug-in für vCenter Server erforderlich.

.Schritte
. Für vCSA-Upgrades https://docs.netapp.com/us-en/vcp/vcp_task_qossioc.html#clear-qossioc-settings["klar"^] QoSSIOC-Einstellungen im Plug-in (* NetApp Element > QoSSIOC-Einstellungen*).  Das Feld *QoSSIOC-Status* zeigt an `Not Configured` nach Abschluss des Prozesses.
. Für vCSA- und Windows-Upgrades, https://docs.netapp.com/us-en/vcp/task_vcp_unregister.html["abmelden"^] Das Plug-in des vCenter Servers, mit dem es über das Registrierungsprogramm verknüpft ist.
. https://docs.vmware.com/en/VMware-vSphere/6.7/com.vmware.vcenter.upgrade.doc/GUID-7AFB6672-0B0B-4902-B254-EE6AE81993B2.html["Aktualisieren Sie vSphere, einschließlich vCenter Server, ESXi, VMs und anderer VMware-Komponenten."^].
+
[IMPORTANT]
====
Sie sollten auf NetApp Element Plug-in für vCenter Server 5.0 oder höher aktualisieren, um die Möglichkeit zu erhalten, das Plug-in mit VMware vCenter 7.0 Update 3 ohne Anwendung eines Workarounds bereitzustellen.

Beim Upgrade auf VMware vCenter Server 7.0 Update 3 schlägt die Bereitstellung des Element-Plug-ins für vCenter Server 4.x fehl. Um dieses Problem mit Spring Framework 4 zu beheben, siehe https://kb.netapp.com/Advice_and_Troubleshooting/Hybrid_Cloud_Infrastructure/NetApp_HCI/vCenter_plug-in_deployment_fails_after_upgrading_vCenter_to_version_7.0_U3["dieser KB-Artikel"^]Die

====
. https://docs.netapp.com/us-en/vcp/vcp_task_getstarted.html#register-the-plug-in-with-vcenter["Anmeldung"^]Das Element-Plug-in für vCenter Server wieder mit vCenter.
. https://docs.netapp.com/us-en/vcp/vcp_task_getstarted.html#add-storage-clusters-for-use-with-the-plug-in["Cluster hinzufügen"^]unter Verwendung des Plug-ins.
. https://docs.netapp.com/us-en/vcp/vcp_task_getstarted.html#configure-qossioc-settings-using-the-plug-in["QoSSIOC-Einstellungen konfigurieren"^]unter Verwendung des Plug-ins.
. https://docs.netapp.com/us-en/vcp/vcp_task_qossioc.html#enabling-qossioc-automation-on-datastores["QoSSIOC aktivieren"^]für alle vom Plug-in gesteuerten Datenspeicher.
