= Ein Element-Speichersystem einrichten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Einrichtung eines NetApp Element Software-Speichersystems umfasst die Konfiguration eines Speicherknotens, die Erstellung eines Speicherclusters und das Hinzufügen von Laufwerken zum Cluster.  Wenn Sie ein Fibre-Channel-Netzwerk verwenden, können Sie einen Fibre-Channel-Knoten konfigurieren.
