= Richten Sie einen Verwaltungsknoten ein.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können den NetApp Element Software Management Node (mNode) installieren, um Systemdienste zu aktualisieren und bereitzustellen, Cluster-Assets und -Einstellungen zu verwalten, Systemtests und -Dienstprogramme auszuführen und den NetApp Support-Zugriff für die Fehlerbehebung zu aktivieren.

. Siehe dielink:../mnode/task_mnode_install.html["Installieren Sie den Management-Knoten"] Dokumentation.
+

NOTE: Um Active IQ einzurichten, verwenden Sie die `--telemetry_active` Parameter im Setup-Skript, um die Datenerfassung für Analysen durch Active IQ zu aktivieren.
