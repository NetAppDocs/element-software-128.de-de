= Einen Snapshot-Zeitplan löschen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können einen Snapshot-Zeitplan löschen.  Nach dem Löschen des Zeitplans werden keine weiteren geplanten Snapshots mehr ausgeführt.  Alle im Rahmen des Zeitplans erstellten Snapshots bleiben auf dem Speichersystem erhalten.

. Klicken Sie auf *Datenschutz* > *Zeitpläne*.
. Klicken Sie auf das Symbol *Aktionen* für den Zeitplan, den Sie löschen möchten.
. Klicken Sie im daraufhin angezeigten Menü auf *Löschen*.
. Bestätigen Sie die Aktion.
