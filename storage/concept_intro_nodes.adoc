= Nodes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Knoten sind die einzelnen Hardwarekomponenten, die zu einem Cluster zusammengefasst werden, um als Blockspeicher genutzt zu werden.  In einem SolidFire -Speichersystem gibt es zwei Arten von Knoten: Speicherknoten und Fibre-Channel-Knoten.
