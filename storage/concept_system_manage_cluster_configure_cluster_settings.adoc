= Clustereinstellungen konfigurieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Über die Registerkarte „Cluster“ der Element-Benutzeroberfläche können Sie clusterweite Einstellungen anzeigen und ändern sowie clusterspezifische Aufgaben ausführen.

Sie können Einstellungen wie Cluster-Auslastungsschwelle, Supportzugriff, Verschlüsselung ruhender Daten, virtuelle Volumes, SnapMirror und NTP-Broadcast-Client konfigurieren.
