= Speicherverwaltung mit der Element-Software
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Mit der Element-Software können Sie SolidFire -Speicher einrichten, die Clusterkapazität und -leistung überwachen und die Speicheraktivität in einer Multi-Tenant-Infrastruktur verwalten.

Element ist das Speicherbetriebssystem, das das Herzstück eines SolidFire Clusters bildet.  Die Element-Software läuft unabhängig auf allen Knoten im Cluster und ermöglicht es den Knoten des Clusters, Ressourcen zu bündeln und sich externen Clients als ein einziges Speichersystem darzustellen.  Die Element-Software ist für die gesamte Clusterkoordination, Skalierung und Verwaltung des Gesamtsystems verantwortlich.

Die Softwareschnittstelle basiert auf der Element API.
