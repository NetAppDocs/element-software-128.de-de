= Führen Sie die Remote-Replikation zwischen Clustern durch, auf denen die NetApp Element -Software ausgeführt wird.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bei Clustern, auf denen die Element-Software läuft, ermöglicht die Echtzeitreplikation die schnelle Erstellung von Remote-Kopien der Volume-Daten.  Sie können einen Speichercluster mit bis zu vier anderen Speicherclustern koppeln.  Sie können Volumendaten synchron oder asynchron von jedem der beiden Cluster eines Clusterpaares für Failover- und Failback-Szenarien replizieren.
