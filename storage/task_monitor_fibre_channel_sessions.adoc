= Fibre Channel-Sitzungen ansehen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die mit dem Cluster verbundenen Fibre Channel (FC)-Sitzungen anzeigen.  Sie können die Informationen filtern, sodass nur die Verbindungen im Fenster angezeigt werden, die Sie sehen möchten.

. Wählen Sie in der Element-Benutzeroberfläche *Berichterstellung* > *FC-Sitzungen*.
. Um die Filterkriterienfelder anzuzeigen, klicken Sie auf *Filter*.
