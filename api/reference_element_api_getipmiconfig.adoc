= GetIpmiConfig
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die `GetIpmiConfig` Methode zum Abrufen von Hardware-Sensorinformationen von Sensoren, die sich in Ihrem Knoten befinden.



== Parameter

Diese Methode hat folgende Eingabeparameter:

|===
| Name | Beschreibung | Typ 


 a| 
Chassis-Typ
 a| 
Dient zur Anzeige von Informationen für jeden Knoten-Chassis-Typ. Mögliche Werte:

* all: Gibt Sensorinformationen für jeden Chassis-Typ zurück.
* {chassis type}: Gibt Sensorinformationen für einen angegebenen Chassis-Typ zurück.

 a| 
Schnur

|===


== Rückgabewerte

Diese Methode hat die folgenden Rückgabewerte:

|===
| Name | Beschreibung | Typ 


 a| 
sensorName
 a| 
Name des gefundenen Sensors.
 a| 
Schnur



 a| 
eindeutigeSensor-ID
 a| 
Eindeutige Kennung für den Sensor.
 a| 
Schnur

|===


== Anforderungsbeispiel

Anfragen für diese Methode ähneln dem folgenden Beispiel:

[listing]
----
{
  "method": "GetIpmiConfig",
  "params": {
     "chassisType"; "all"
     },
  "id" : 1
}
----
