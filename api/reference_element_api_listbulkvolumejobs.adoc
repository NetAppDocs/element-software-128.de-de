= ListBulkVolumeJobs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die `ListBulkVolumeJobs` Methode zum Erhalten von Informationen über jeden einzelnen Lese- oder Schreibvorgang eines Massendatenträgers, der im System stattfindet.



== Parameter

Diese Methode hat keine Eingabeparameter.



== Rückgabewert

Diese Methode hat folgenden Rückgabewert:

|===
| Name | Beschreibung | Typ 


 a| 
Massenvolumenaufträge
 a| 
Eine Reihe von Informationen für jeden Massenauftrag.
 a| 
xref:reference_element_api_bulkvolumejob.adoc[Massenvolumenauftrag]Array

|===


== Anforderungsbeispiel

Anfragen für diese Methode ähneln dem folgenden Beispiel:

[listing]
----
{
   "method": "ListBulkVolumeJobs",
   "params": {
      },
   "id" : 1
}
----
