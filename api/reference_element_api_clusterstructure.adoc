= Clusterstruktur
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das clusterStructure-Objekt enthält Sicherungsinformationen zur Clusterkonfiguration, die mit der GetClusterStructure-Methode erstellt wurden.  Mit der Methode SetClusterStructure können Sie diese Informationen in einem Speichercluster wiederherstellen, den Sie gerade neu aufbauen.
