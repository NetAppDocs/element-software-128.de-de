= fipsErrorNodeReport
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das Objekt fipsErrorNodeReport enthält Fehlerinformationen für jeden Knoten, der bei einer Abfrage keine Informationen zur FIPS 140-2-Unterstützung liefert. `GetFipsReport` Verfahren.



== Objektmitglieder

Dieses Objekt enthält die folgenden Elemente:

|===
| Name | Beschreibung | Typ 


 a| 
Knoten-ID
 a| 
Die ID des Knotens, der nicht geantwortet hat.
 a| 
ganze Zahl



 a| 
Fehler
 a| 
Ein JSON-Objekt, das Fehlerinformationen enthält.
 a| 
JSON-Objekt

|===
