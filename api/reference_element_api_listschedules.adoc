= Zeitpläne auflisten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können verwenden `ListSchedules` Informationen über alle erstellten geplanten Snapshots erhalten.



== Parameter

Diese Methode hat keine Eingabeparameter.



== Rückgabewert

Diese Methode hat folgenden Rückgabewert:

|===
| Name | Beschreibung | Typ 


 a| 
Zeitpläne
 a| 
Eine Liste der aktuell auf dem Cluster laufenden Zeitpläne.
 a| 
xref:reference_element_api_schedule.adoc[Zeitplan]Array

|===


== Anforderungsbeispiel

Anfragen für diese Methode ähneln dem folgenden Beispiel:

[listing]
----
{
  "method": "ListSchedules",
    "params": {},
"id": 1
}
----
