= DisableBmcColdReset
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die `DisableBmcColdReset` Methode zum Deaktivieren der Hintergrundaufgabe, die den Baseboard Management Controller (BMC) für alle Knoten im Cluster periodisch zurücksetzt.



== Parameter

Diese Methode hat keinen Eingabeparameter.



== Rückgabewerte

Diese Methode hat folgenden Rückgabewert:

|===
| Name | Beschreibung | Typ 


 a| 
cBmcResetDurationMinutes
 a| 
Gibt die Zeit zwischen den Rücksetzintervallen zurück.  Das Intervall sollte nach Abschluss des Befehls immer 0 sein.
 a| 
ganze Zahl

|===


== Anforderungsbeispiel

Anfragen für diese Methode ähneln dem folgenden Beispiel:

[listing]
----
{
  "method": "DisableBmcColdReset",
  "params": {},
  "id" : 1
}
----


== Antwortbeispiel

Diese Methode liefert eine Antwort, die dem folgenden Beispiel ähnelt:

[listing]
----
{
    "id": 1,
    "result": {
        "cBmcResetDurationMinutes": 0
    }
}
----
