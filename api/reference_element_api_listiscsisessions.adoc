= ListISCSISessions
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die `ListISCSISessions` Methode zum Auflisten von iSCSI-Verbindungsinformationen für Volumes im Cluster.



== Parameter

Diese Methode hat keine Eingabeparameter.



== Rückgabewert

Diese Methode hat folgenden Rückgabewert:

|===
| Name | Beschreibung | Typ 


 a| 
Sitzungen
 a| 
Informationen zu jeder iSCSI-Sitzung.
 a| 
xref:reference_element_api_session_iscsi.adoc[Sitzung]

|===


== Anforderungsbeispiel

Anfragen für diese Methode ähneln dem folgenden Beispiel:

[listing]
----
{
   "method": "ListISCSISessions",
   "params": {},
   "id" : 1
}
----
