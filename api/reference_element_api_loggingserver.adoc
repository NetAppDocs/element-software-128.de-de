= Protokollierungsserver
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Das loggingServer-Objekt enthält Informationen über alle für den Speichercluster konfigurierten Logging-Hosts.  Sie können verwenden `GetRemoteLoggingHosts` um die aktuellen Protokollierungshosts zu ermitteln und dann zu verwenden `SetRemoteLoggingHosts` um die gewünschte Liste aktueller und neuer Logging-Hosts festzulegen.



== Objektmitglieder

Dieses Objekt enthält die folgenden Elemente:

|===
| Name | Beschreibung | Typ 


 a| 
Gastgeber
 a| 
IP-Adresse des Log-Servers.
 a| 
Schnur



 a| 
Hafen
 a| 
Portnummer, die zur Kommunikation mit dem Log-Server verwendet wird.
 a| 
ganze Zahl

|===
