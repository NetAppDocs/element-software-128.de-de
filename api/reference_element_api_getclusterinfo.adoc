= GetClusterInfo
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die `GetClusterInfo` Methode zur Rückgabe von Konfigurationsinformationen über den Cluster.



== Parameter

Diese Methode hat keine Eingabeparameter.



== Rückgabewert

Diese Methode hat folgenden Rückgabewert:

|===
| Name | Beschreibung | Typ 


 a| 
Clusterinformationen
 a| 
Clusterinformationen.
 a| 
xref:reference_element_api_clusterinfo.adoc[Clusterinformationen]

|===


== Anforderungsbeispiel

Anfragen für diese Methode ähneln dem folgenden Beispiel:

[listing]
----
{
   "method": "GetClusterInfo",
   "params": {},
   "id" : 1
}
----
