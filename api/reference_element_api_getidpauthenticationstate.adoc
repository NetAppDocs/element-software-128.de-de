= GetIdpAuthenticationState
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die `GetIdpAuthenticationState` Methode zur Rückgabe von Informationen über den Status der Authentifizierung unter Verwendung von Drittanbieter-IdPs.



== Parameter

Diese Methode hat keine Eingabeparameter.



== Rückgabewerte

Diese Methode hat folgenden Rückgabewert:

|===
| Name | Beschreibung | Typ 


 a| 
ermöglicht
 a| 
Zeigt an, ob die Authentifizierung durch Drittanbieter-Identitätsanbieter aktiviert ist.
 a| 
boolescher Wert

|===


== Anforderungsbeispiel

Anfragen für diese Methode ähneln dem folgenden Beispiel:

[listing]
----
{
  "method": "GetIdpAuthenticationState"
}
----


== Antwortbeispiel

Diese Methode liefert eine Antwort, die dem folgenden Beispiel ähnelt:

[listing]
----
{
  "result": {"enabled": true}
}
----
